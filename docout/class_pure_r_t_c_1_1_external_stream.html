inlinestatic

This factory returns a PureRTC.ExternalStream instance with user defined options.



options:

  • url: RTSP stream URL
  • video: boolean or object. Default is false. If the value is a boolean, it indicates whether video is enabled or not. If the value is an object, it may have device property ,e.g.,{device: 'camera'} indicates the stream is from camera.
  • audio: true/false. Default is false.
  • Note:if both video and audio are false or undefined, it will fail to publish an ExternalStream.


callback:
Upon success, err is null, and externalStream is an instance of PureRTC.ExternalStream; upon failure externalStream is undefined and err is one of the following:

  • {code: 1107, msg: 'USER_INPUT_INVALID'} â€“ user input media option is invalid.
Parameters
optionsStream creation options.The options must have url property and must have video or audio.
callbackcallback(err, externalStream) will be invoked when ExternalStream creation is done.

<script type="text/javascript">
// ExternalStream
var externalStream;
PureRTC.ExternalStream.create({
url: 'http://www.example.com/camera',
video: true,
audio: true
}, function (err, stream) {
if (err) {
return console.log('create ExternalStream failed:', err);
}
externalStream = stream;
});
</script>
