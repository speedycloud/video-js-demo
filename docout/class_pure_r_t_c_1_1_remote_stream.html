Detailed Description

Stream from server retrieved by 'stream-added' event.

RemoteStreams are automatically constructed upon the occurrence of the event.
Example:

<script type="text/javascript">
conference.on('stream-added', function (event) {
var remoteStream = event.stream;
console.log('stream added:', stream.id());
});
</script>

This function removes listener(s) for a specified event.

If listener is unspecified, all the listener(s) of the event would be removed; or if the listener is in the event listener list, it would be removed; otherwise this function does nothing.
